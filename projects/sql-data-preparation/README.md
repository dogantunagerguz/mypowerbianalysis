# SQL Data Preparation

This project shows SQL fundamentals used to prepare structured data for BI reporting and analysis.

## Focus

- Relational data thinking
- JOIN logic
- Aggregations
- Filtering and validation
- Reporting-ready outputs
- Clear query documentation

The sample query uses synthetic customer, sales and payment tables. It is intentionally generic and contains no real company or customer data.
