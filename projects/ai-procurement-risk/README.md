# AI-Supported Risk Detection in Public Procurement Documents

Master's thesis project focused on using Python, NLP and large language model prompting to identify commercial risk indicators in public procurement and tender documents.

## Problem

Procurement documents contain long, semi-structured text. Important commercial or contractual risk indicators can be difficult to identify consistently through manual review.

## Approach

1. Prepare procurement document text.
2. Define risk categories and structured output fields.
3. Create prompts for GPT-based and Gemini-based models.
4. Compare model outputs against the evaluation criteria.
5. Use BERTScore-based evaluation to assess output consistency.
6. Document limitations and human-review requirements.
